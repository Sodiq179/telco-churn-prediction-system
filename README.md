# telco-churn-prediction-system

End-to-end ML project with training and deployment pipeline.

## Structure
- src/ → core code
- configs/ → configuration
- data/ → datasets
- artifacts/ → outputs
- scripts/ → automation
- tests/ → testing

## Setup
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```
